## DESKTOP WALLETS

### Sparrow Wallet
- FAQ oficial: https://sparrowwallet.com/docs/faq.html
- Sigue BIP39 estricto: no almacena passphrase en archivo wallet
- Master Fingerprint visible al importar seed + passphrase

### Electrum
- Docs seed: https://electrum.readthedocs.io/en/latest/seedphrase.html
- Importar Ledger/Trezor: https://coinguides.org/import-ledger-trezor-seed-electrum/
- Llama a la passphrase "extension word" o "custom words" (no BIP39 nativo)
- Para usar BIP39: Options → BIP39 seed → "Extend with custom words"

### Specter Desktop
- Docs: https://docs.specter.solutions/desktop/faq/
- GitHub: https://github.com/cryptoadvance/specter-desktop
- Campo "BIP39 passphrase (optional)" en diálogo de setup de dispositivo

### Blockstream Green (Desktop + Mobile)
- Setup passphrase: https://help.blockstream.com/hc/en-us/articles/8721715194649-Set-up-a-BIP39-passphrase
- Añadido en v2.0.4 (2024)
- Acceso: ⋯ (tres puntos) → "Login with BIP39 Passphrase"

## HOT WALLETS MOBILE

### BlueWallet
- Import wallet docs: https://bluewallet.io/docs/import-wallet/
- GitHub wallets soportados: https://github.com/BlueWallet/BlueWallet/wiki/Supported-wallet-types
- Requiere: Advanced Mode activado para ver campo passphrase

### Bull Bitcoin Wallet
- GitHub: https://github.com/SatoshiPortal/bullbitcoin-mobile
- Bitcoin + Lightning only | passphrase almacenada en secure storage del dispositivo

### Phoenix Wallet
- Sitio: https://phoenix.acinq.co/
- GitHub: https://github.com/ACINQ/phoenix
- Lightning-only — NO soporta BIP39 passphrase standard
- Usa sistema propio de seed para canales Lightning

### Blockstream Green Mobile
- Misma UX que desktop (v2.0.4+)
- Login BIP39 Passphrase desde ⋯ en pantalla principal

### Nunchuk
- Sitio: https://nunchuk.io/
- Recursos: https://resources.nunchuk.io/wallet-recovery/personal-wallet/
- Botón "[Set passphrase]" al añadir software key
- Bitcoin-only, multisig-focused
- Fingerprint actualiza al añadir passphrase (verificación visual)
